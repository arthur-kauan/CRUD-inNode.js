# CRUD-inNode.js
Repositório para entrega da atividade "Entrega do Trabalho API Rest completa em Node.js" do professor Tiago Dutra Galvão.

Nesse repositório, serão postados commits da atividade de uma API completa e funcional em Node.js.
Com funcionalidade de armazenar dados localmente em um array e sendo capaz de alterá-los, criá-los, deletá-los e escreve-los em tela.
Meu tema escolhido foram jogos, especialmente os últimos que joguei, mas podendo haver jogos que ainda desejo jogar.
